# MABI Landing Page — Update after Phase 1-8

## Context
We've massively expanded MABI today. The landing page needs to reflect the new capabilities. Current page has 12 modules but doesn't show the key differentiators we've built.

## Key messaging changes

### Old narrative: "ERP replacement with modules"
### New narrative: "AI-powered platform that thinks, watches, and acts"

The biggest differentiator is NOT that we have PIM + CRM + WMS. Everyone has modules.
The differentiator is:
1. **AI Brain** — system watches everything and surfaces what matters
2. **One flow, not silos** — scan invoice → asset created → maintenance scheduled → budget updated → all automatic
3. **Mobile-first field ops** — sales reps and warehouse workers have real apps
4. **Polish-first** — KSeF, KŚT, JPK, Polish accounting, PLN/EUR

## Updated Module List (16 modules now)

1. **PIM** — Product Information Management + AI completeness scoring + publication workflow
2. **MES** — Manufacturing Execution + Production TV Dashboard (live shop floor screens)
3. **WMS** — Warehouse Management + PWA Terminal (PDA scanners) + real-time stock
4. **CRM** — Customer Relationship + VoIP/Twilio + Email campaigns + Sequences + Automations
5. **Finance** — Full accounting: invoices, journal, bank reconciliation, reports, fixed assets, depreciation, budgets with approval workflow
6. **Procurement** — Purchase Requests → Orders → 3-way invoice matching. Budget checked BEFORE you spend.
7. **AI Hub** — LangGraph agents, 6 watchers (mail, stock, maintenance, budget, delivery, PIM), suggestion panel
8. **e-Commerce** — B2C/B2B storefront powered by PIM data
9. **B2B Platform** — Customer portal: catalog, orders, documents, account
10. **DMS** — Document management: versioning, expiry alerts, approval, ISO 9001 ready
11. **QR Maintenance** — CMMS: assets, schedules, spare parts, QR codes, TV dashboard
12. **Import Hub** — Supplier sourcing: containers, shipments, QA, cost calculation
13. **Field Sales** — Mobile PWA: catalog, orders, visit reports, GPS map, offline, AI suggestions
14. **Quality Control** — QC records, inspections, defect tracking
15. **Projects** — Task management, boards, collaboration
16. **KSeF** — Polish e-invoice system integration (send & receive)
